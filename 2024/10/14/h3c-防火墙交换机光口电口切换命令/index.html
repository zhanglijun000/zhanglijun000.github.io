H3C 防火墙交换机光口电口切换命令

发表于 2024-10-14 更新于 2024-11-09 分类于 其他知识

Login: admin #输入账号
Password: #输入密码
%Jan 1 00:30:23:827 2011 H3C SHELL/5/SHELL_LOGIN: admin logged in from con0.

sys #进入配置模式
System View: return to User View with Ctrl+Z.
[H3C] int g 1/0/0 #进入接口0命令
[H3C-GigabitEthernet1/0/0]dis this #显示当前端口状态

interface GigabitEthernet1/0/0
port link-mode route
combo enable copper # copper为当前端口为电口
ip address 192.168.0.1 255.255.255.0

return
[H3C-GigabitEthernet1/0/0]combo enable fiber #fiber 调整为光口模式
