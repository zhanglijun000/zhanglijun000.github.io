联想M7206 M7216 LT201 加粉后打印机提示无粉，清零方法详解联想LT201 TN1035/TN1000/LT201/CT2021138粉盒复位方法 M2040 加粉 复位 清零 方法

发表于 2022-07-31 更新于 2024-11-09 分类于 打印机知识

【机器型号】联想M7206打印机

【粉盒型号】LT201

【故障现象】加粉后打印机提示无粉

【维修过程】

　　加粉后依然提示无粉，2种清零方法

①齿轮清零

取下双联齿轮

粉用尽时刻，齿轮位置如上图。

初始位置，清零后的位置。白色弹片，卡在凹槽里面。再来一张高清图。

复位齿轮运行轨迹：

　这个运动轨迹是对的，想法是错误的。

　　我用手转磁辊，6-7周，复位齿轮就从起始位置跳到了粉用尽位置。显然，无粉，是打印机判断的，这复位齿轮，就起始位置有一定作用。

百度搜了一下，有2个答案比较靠谱：

A、复位齿轮起到打印机检测的作用，不复位，打印机检测不到粉盒。也就是说：

1.新粉盒不用管复位不复位，只有加了粉之后才会用到。
2.复位是起到让打印机可以检测到，否则加了粉，打印机也检测不到

B、如果齿轮不复位的话机器是检测不到硒鼓的！

相当一个原位只有到了那个位置机器才能正常工作。

从外面看的复位齿轮是个半圆形的白色顶盖。复位之前是左白右黑，上机后，随着齿轮咬合转动，迅速转到了左黑又白的位置，就是粉用完 的位置。

　　复位齿轮转的大半圈，应该是带动机器上的一个齿轮运动的机器上的齿轮，会触发一个检测到硒鼓的信号给机器。这都是我的假想，具体什么原理，以后我会拆一台兄弟打印机看看。

②按键复位方法：

　　嫌拆卸齿轮麻烦的话，可以手工按几个按键复位。

功能键>设备信息>OK 键>重置硒鼓>长按OK键>屏提示重置>开始键>息屏（无屏显）>按向上箭头按到“11”>OK键>OK键

HP A3打印机 HP436N 更换载体，重置载体

发表于 2022-07-31 更新于 2024-11-09 分类于 打印机知识

1，开机就绪状态下，开始按照下面的顺序进行按键：菜单#10043616菜单（请快速、连续完成操作，否则无法进入tech mode菜单）。
2,操作控制面板进入Tech menu并选择Service Functions下面的TC Init。
3,打印机提示：Initialize Now?

重装系统后没有打开文件的权限

发表于 2022-07-31 更新于 2024-11-09 分类于 电脑知识

1、点击“安全选项卡”，进入文件夹属性中的安全栏。

2、点击高级，进入“高级安全设置”，点击更改。

3、进入“选择用户或组”，点击左下角“高级”，点击“立即查找”，出现一批用户名。

4、选择你现在的管理员用户名，点击确定。

5、把下面的“替换子容器和对象的所有者”勾上，点击确定，一直确定。

6、最终出现“WINDOWS安全”，点击确定。

7、最后重新打开文件夹即可访问了。
