网络共享打印机连接方法

发表于 2022-08-06 更新于 2024-11-09 分类于 打印机知识 ， 电脑知识

首先在已经安装好驱动的电脑上设置打印机网络共享

在开始界面打开设备和打印机，找不到就搜索一下，或者找或者搜索控制面板，里面找《设备和打印机》，选择要共享的打印机，右键>打印机属性（注意不是属性，是打印机属性）。

勾选共享打印机>应用>确定。

然后设置一下网络共享设置，在电脑右下角找到电脑图标，右键>打开网络和共享中心

打开高级共享设置，按照下图设置

设置完成后，在电脑属性里找到计算机名，记下来。或者开始>运行>cmd>ipconifg

通过计算机名和电脑IP地址其实都可以连接网络共享打印机，但是不建议使用电脑IP来连接。因为大部分网络中的电脑IP地址并不是固定的，如果电脑的IP地址发生变化，共享打印机就不能用了。而计算机名除做系统外一般是不会变的。接下来就可以在要链接共享打印机的电脑上来找打印机了。

在桌面上找到《网络》，没有的话就在计算机的左侧找一下，打开要等一下，电脑会自动搜索网络中的计算机。

搜索到咱们共享打印机的电脑，双击打开就可以，如果网络共享设置没有设置好就会报错，正常的话就会看到咱们已经共享好的打印机

找到共享的打印机后，正常双击就可以用了，如果连接共享打印机电脑系统与共享打印机电脑系统不一致，驱动兼容就会报错，这样就需要咱们在要连接的电脑上先安装好打印机驱动在进行网络连接就可以了。

在这之前先教一下大家另外两种找到共享打印机电脑的方法，因为有时候在网络里面是搜索不到另外一台电脑的，这样就需要咱们在运行里面直接去找共享打印机的电脑。方法也非常简单，直接在开始里面运行，或者按键盘上的Win+R键打开运行窗口，在里面输入\\计算机名（或者电脑IP）,然后直接运行就可以到达共享打印机界面。

这里接下来说安装打印机驱动，在浏览器里面搜索找到打印机驱动网，或者直接输入打印机驱动网网址http://www.dyjqd.com/

这是我的打印机驱动，自己打印的驱动请按各自的型号下载，下载完成后直接安装就可以，选择自动安装>下一步直至完成就可以了。

安装完成后就可以接着在网络电脑中添加打印机了，IP地址和计算机名在设备和打印机里面显示不一样的，按照需要设置默认打印机即可。

三.跳线设置说明

1.J1跳线设置说明

①.J1跳1-2脚[Lock]: 电锁信号输出.

NO和LOCK-接通电开锁(如电控锁).

NC和LOCK-接断电开锁(如电插锁或磁力锁).

特别提示:为保证门禁运行稳定，建议电流大于500MA的电锁不要直接从门禁接线

②.J1跳2-3脚[Null]: 干触点信号输出.

NO和COM输出常开信号.

NC和COM输出常闭信号.

③.J1跳3-4脚[Push]: 出厂默认设置,电源信号输出,电锁接门禁专用电源,有效信号PUSH输出低电平.

2.J3跳线设置说明

①.J3跳线跳1-2脚[Single]:出厂默认设置,之本机读卡。

②.J3跳线跳2-3脚[Double]:本机和外接分体读卡头可双向读卡。

四．单门门禁一体机系统编程设置方法：

初始管理密码为12345, 用户密码为8888

1.进入编程状态:

按* => 输入管理密码 =>按#

2 功能设置(需在编程状态下)

(1) 修改管理密码

按0 => 输入新管理密码 => 按# => 再输入一次新管理密码 =>按#=>按*

注:密码为4-8 位任意数字

(2) 增加用户卡

按1 => 读卡 => 输入该卡四位数代码 => 连续读卡 => 按#=>按*

注:①.卡的代码为0001 - 9999任意不可重复的四位数字,

②.第一张卡必须输入代码否则该卡无效

③.如需连续增加卡则在输完四位数的代码后不必按#再输代码，直接连续读卡,最后按# 键

(3) 删除用户卡有三种方法

①.删除全部用户卡: 按2 => 按0000 => 按 #=>按*

②.删除被读的卡: 按2 => 读卡(如多张卡则连续读卡) =>按#=>按*

③.删除遗失的卡: 按2 => 该卡代码 =>按#=>按*

(4) 设置开门模式

①.读有效卡开门: 按3 0 0 => 按# =>按*

②.读有效卡加密码开门: 按3 0 1 => 按#=>按*

③.读有效卡开门或者输入正确密码开门:按3 0 2 => 按#=>按*

注:出厂设置为302读有效卡或者输入密码开门

(5) 设置开门时间

按4 => X X => 按#=>按*
