震旦219柯美1657723184系列维修召唤C2558_C2557维修呼叫C2558 故障代码

发表于 2022-07-31 更新于 2024-11-09 分类于 打印机知识

现象：机器跳C2558 碳粉浓度过浓故障代码

原因及解决方案：

1、 显影仓内载体耗尽，更换载体并重做初始化 进入维修模式，通过方向键选中“FUNCTION”并点击“OK”键进入。 取出粉瓶，通过方向键选中“TCR AUTO ADJUST”并点击“OK”键开始初始化。

初始化完毕，屏幕如下右图显示，并给出显影仓内当前的碳粉浓度。（正常在 145 左右） 初始化过程将近 5 分钟左右，请耐心等待。

2、 TCR 传感器连线未接上，接上后重启测试

soso_tc_slider_img

3、 浓度确实过高，取出粉瓶，拉黑板降低浓度

1.进入维修模式（效用 — 停止 — 0 — 0 — 停止 — 0 — 1），

通过方向键选中“DISPLAY”并点击“OK”键进入。

通过方向键选中“TCR GAIN”并点击“OK”键进入。

查看当前机器 TCR 值。

2.通过方向键选中“ADJUST”并点击“OK”键进入。

通过方向键选中“TCR GAIN”并点击“OK”键进入。

(按 3 次上键） 通过方向键，选择比当前 TCR 值略高的数值，并点击“OK”键确认。

3.关闭电源并耐心等待 10 秒，重新开启复印机。若仍跳代码，按以上步骤继续调高 TCR GAIN 的数值，直至复印机正常开启。 放入适量纸张（建议使用再生纸），取出碳粉瓶，打开稿 台盖板，连续复印10张全黑件。

4 再次查看设备当前 TCR 值，并调底 ADJUST 中的 TCR GAIN 值，继续如上动作，直至 设备 当前 TCR 值回到 145 左右，并且不跳代码。

这个代码是复印机最常见的，希望大家仔细的研究。

没有数字按钮？

请取出粉瓶、打开前门和则门——–取出碳粉——–拆下套鼓线束盖板———拔下线束——-拆下两颗固定螺丝（保右门是打开状态）———平行取出套鼓——–然后平行摇动套鼓（以使载体能恢复正常水平）—–装回机器测试，现在只能往上调，关机10秒后开机如果不出代码，复印20张左右：【按菜单／选择键】【快速设置】【←】【←】【快速设置】【←】【→】【FUNCTION】【TCR AUTO ADJUST】——–,然后圧开始它自动调整，就要复印整版黑色，请按以下方法操作，【按菜单／选择键】【快速设置】【←】 【←】【快速设置】【←】【→】——-进入维修模式——–在维修模式功能中改变设置值【ADJUST】【TCR】增益【TCR GAIN】【 ADJUST】【TCR GAIN】——–用向上或向下的箭头调整里面的数值。再次进维修模式。

如果第一种方案不能解决问题，将载体里面的墨粉先复印掉： 依次按以下按键。 （·默认设詈为140） 一般调到165，然后按以下方法操作【按菜单／选择键】【快速设置】【←】【←】【快速设置】【←】【→】
